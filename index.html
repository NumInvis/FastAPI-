FastAPI 教程

你懂 Python 语法，但从未写过后端？
这门课从"浏览器如何访问服务器"开始，手把手带你做出生产级的 Web API。

Python 3.10+ 零基础后端 项目实战

这门课怎么教你？

不是罗列 API，而是用故事场景引入每个知识点：

1

场景引入

先给你一个真实的业务场景（比如"用户注册时要检查邮箱格式"），让你明白学这个东西是为了解决什么问题。

2

原理图解

用可视化流程图展示背后的运行机制，不让你死记硬背。

3

代码实战

一步一步写出代码，同时展示"新手常犯的错误"和"正确的写法"做对比。

4

交互测验

每章末尾都有选择题和代码分析题，答错会告诉你为什么，巩固记忆。

第一个 API：从请求到响应

后端到底是什么？HTTP 请求如何进入你的代码？GET、POST、路径参数、查询参数、JSON 响应。
基础篇 · 02

数据校验：让数据安全进门

用户传什么你都得信吗？Pydantic 帮你自动检查数据类型、长度、范围，错误时返回友好提示。
基础篇 · 03

依赖注入：代码的乐高积木

如何避免到处复制粘贴相同的代码？学会用依赖把通用逻辑（如数据库连接）抽象成可复用的积木。
进阶篇 · 04

数据库：数据的持久化家园

数据不能只存在内存里。SQLAlchemy 带你优雅地操作数据库，不用手写 SQL。
进阶篇 · 05

登录与权限：守门人与通行证

密码怎么存才安全？JWT 令牌是什么？如何实现"只有管理员能删除"的权限控制？
进阶篇 · 06

错误处理与日志：事故的善后

代码出错了怎么办？如何给前端返回统一的错误格式？怎么记录日志方便事后排查？
进阶篇 · 07

异步与后台任务：不等人的艺术

发邮件为什么不能让用户等？async/await 到底在干什么？Celery 任务队列初探。
进阶篇 · 08

WebSocket：实时对话的桥梁

HTTP 是一次性对话，WebSocket 是持续通话。实现聊天室、实时通知推送。
部署篇 · 09

测试：在上线前发现问题

手动刷新浏览器测试太蠢了。pytest 自动帮你测试每个接口，确保重构不破坏旧功能。
部署篇 · 10

性能优化与部署：走向世界

缓存、限流、Docker 容器化、Nginx 反向代理。让你的服务从笔记本走向云服务器。
部署篇 · 11

企业级架构：像专家一样组织代码

项目大了代码怎么放？Clean Architecture 分层、健康检查、监控告警。毕业项目。
开始学习前的准备

你需要：安装 Python 3.10+、一个代码编辑器（VS Code 或 PyCharm）、一颗愿意动手实践的心。不需要懂前端，不需要懂数据库，不需要懂网络协议——这些都会在课程中讲到。
