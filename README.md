## 🖥️ 本地运行（任选一种）

### 方式一：直接打开（最简单）
```bash
# 克隆仓库
git clone https://github.com/NumInvis/FastAPI-.git
cd FastAPI-

# 双击 index.html 即可在浏览器中打开课程
```

### 方式二：本地 HTTP 服务器（推荐，支持代码高亮完整功能）
```bash
# Python 3
cd FastAPI-
python -m http.server 8080

# 然后打开 http://localhost:8080
```

或使用 VS Code 的 **Live Server** 插件一键启动。

## 🎨 功能特色

- **📱 响应式设计**：手机、平板、电脑都能舒适阅读
- **🌗 深色/浅色主题**：一键切换，护眼友好
- **🗂️ 侧边栏导航**：快速跳转任意章节，学习进度一目了然
- **💻 代码高亮**：基于 highlight.js，支持 Python / Bash / JSON 语法高亮
- **🎯 交互式图解**：流程图、架构图帮助理解抽象概念
- **📦 纯静态**：零后端依赖，托管到任意静态服务器即可运行
